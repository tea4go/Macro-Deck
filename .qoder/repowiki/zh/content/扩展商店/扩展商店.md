## 简介
本文档面向 Macro-Deck 扩展商店的使用者与开发者，系统化阐述扩展发现机制、安装流程与版本管理策略；详解 ExtensionStoreHelper 的实现与扩展商店的 UI 组成；记录扩展元数据结构、版本信息与依赖关系；解释在线发现与离线安装能力；提供扩展安装、更新与卸载的完整流程；说明扩展与插件系统的集成关系；并给出安全校验与签名机制的现状说明，以及面向开发者的发布指导与用户的扩展管理工具使用建议。

**更新** 基于最新的 XML 文档增强，扩展了 ExtensionStoreHelper 类的详细注释，提供了更清晰的扩展管理流程说明。

## 核心组件
- **扩展商店辅助器（ExtensionStoreHelper）**：负责扩展安装入口、批量更新检查、触发下载对话框、统一事件通知等。新增了详细的 XML 文档注释，涵盖完整的安装流程说明。

## 架构总览
扩展商店采用"UI 视图 + 辅助器 + 下载器 + 模型"的分层设计：
- **UI 层**：ExtensionStoreView 负责在线浏览与搜索；InstalledExtensionsView 负责本地扩展管理。
- **控制层**：ExtensionStoreHelper 提供安装、更新检查、批量更新等控制逻辑，包含详细的 XML 文档注释。
- **数据层**：通过 Constants 中的 API 基础地址访问扩展商店 REST 接口，返回 ApiV2* 结构。
- **安装层**：ExtensionStoreDownloaderItem 负责下载 ZIP、校验 SHA-256、解析 ExtensionManifest 并调用插件或图标包安装器。

### 扩展元数据与清单模型
**更新** 基于增强的 XML 文档注释，提供了更详细的模型说明：

- **ExtensionManifestModel**：扩展清单，包含类型、名称、作者、仓库、包 ID、版本、目标 API、主 DLL 等字段，支持从文件或 ZIP 流反序列化。
- **ApiV2Extension / ApiV2ExtensionFile / ApiV2ExtensionSummary**：扩展商店 API 返回的扩展摘要、文件信息与详细信息。
- **ExtensionStoreDownloaderPackageInfoModel**：扩展商店下载器包信息模型，用于指定要下载安装的扩展包。
- **ExtensionStoreExtensionModel**：扩展商店扩展模型，对应扩展商店 API 返回的扩展信息。
- **PagedList**：分页容器，包含 Items、Page、PageSize、TotalItems。

## 故障排查指南
**更新** 基于增强的安全验证机制，提供了更详细的故障排查：

- **安装失败（哈希不匹配）**：检查日志中 SHA-256 对比失败信息，确认网络下载完整性。
- **下载中断**：查看取消令牌与进度 UI，确认是否被用户取消或页面切换导致请求终止。
- **无法连接商店**：确认 Constants 中的 API 基址可达，检查网络代理与防火墙设置。
- **更新检查无结果**：确保已安装扩展在商店中有对应包 ID 且未被隐藏；检查 ExtensionStoreHelper 的更新检查逻辑。
- **卸载后仍残留**：确认插件管理器或图标管理器的删除流程执行成功，必要时手动清理临时目录。
- **安装包格式错误**：检查 ExtensionManifest.json 是否正确，确保包含必需字段。
