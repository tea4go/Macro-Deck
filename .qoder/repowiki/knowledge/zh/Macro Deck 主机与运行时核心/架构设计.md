- 入口与生命周期：Program.cs 负责单实例检测（命名管道 MacroDeckPipeClient/PipeServer）、路径初始化（ApplicationPaths）与 Serilog 日志引导，随后委托 MacroDeck.Start 完成启动流程。
- 服务器层：MacroDeckServer 基于 ASP.NET Core 内嵌宿主（ServerStartup + MacroDeckServerHelper）暴露 WebSocket 通道（WebSocketHandler/Controller）与静态 Web 客户端（wwwroot/client），并通过 BroadcastServer/AdbServerHelper 支持广播与 ADB 设备桥接。
- 领域管理层：DeviceManager/ProfileManager/Folder/ActionButton 管理设备、配置文件、文件夹与按钮动作执行；EventManager/HotkeyManager/VariableManager/NotificationManager/IconManager/ExtensionStoreHelper 提供事件、热键、变量、通知、图标包与扩展商店能力。
- GUI 层：Windows Forms/WPF 混合界面（MainWindow + 大量 CustomControls/Dialogs/Views），通过 SynchronizationContext 与主线程同步，托盘图标（TrayIconSetup）提供最小化运行与快捷操作。
- 扩展接口边界：IMacroDeckExtension 定义插件/图标包扩展契约，InternalPlugins 提供内置功能插件，外部插件通过 NuGet API 包在运行时由 PluginManager 加载。
