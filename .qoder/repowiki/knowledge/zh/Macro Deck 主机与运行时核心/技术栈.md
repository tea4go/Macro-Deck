- .NET Windows 桌面应用同时启用 WPF 与 Windows Forms（MacroDeck.csproj：UseWPF/UseWindowsForms），并内嵌 ASP.NET Core 宿主（FrameworkReference: Microsoft.AspNetCore.App）用于 WebSocket 与静态文件服务。
- 日志与错误上报采用 Serilog 生态 + Sentry.Serilog（MacroDeck.csproj），模板引擎使用 Cottle（CottleIntegration/TemplateManager）。
- 图像与二维码处理依赖 Magick.NET-Q16-x64/Magick.NET.SystemDrawing 与 QRCoder；移动端桥接通过 AdvancedSharpAdbClient 集成 ADB 工具链（setup/Android Debug Bridge）。
