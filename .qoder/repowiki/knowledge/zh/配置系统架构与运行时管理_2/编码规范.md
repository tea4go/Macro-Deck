- 核心业务逻辑类（如 `MacroDeck`, `MainConfiguration`）普遍采用静态单例或静态属性暴露全局状态，以简化跨模块访问。
- 配置项属性通过 `[JsonProperty]` 特性映射到嵌套风格的 JSON 键名（如 `Connection.Host.Port`），并在 Setter 中处理副作用（如注册表操作）。
- 所有文件系统路径必须通过 `ApplicationPaths` 类的静态属性获取，严禁在业务代码中硬编码目录或文件路径。
- 涉及 UI 交互或窗口创建的操作（如 `ShowMainWindow`）必须通过 `SynchronizationContext` 调度到 UI 线程执行，以确保线程安全。
